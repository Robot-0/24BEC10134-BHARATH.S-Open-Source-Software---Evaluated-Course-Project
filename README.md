# 24BEC10134-BHARATH.S-Open-Source-Software---Evaluated-Course-Project
This repository contains a suite of five custom bash shell scripts designed to perform a comprehensive technical audit of the Python 3 Open Source project. The goal of this audit is to verify the installation integrity, security permissions and operational footprint of Python within a Linux environment.

identity.sh functions as the System Identity while using environment variables.

inspector.sh functions as the Package Audit while using Case statements and License checks.

auditor.sh functions as the Directory Audit while using for loops, stat permissions and du disk usage.

logger.sh functions as the Log Intelligence while using while loops, input redirection and grep filtering.

manifesto.sh functions as the User Interaction while using read command, Here-Documents and File redirection.

INSTALLATION AND USAGE

To replicate this audit on any Debian-based Linux system:

1. Clone the repository using this commmand:

    git clone https://github.com/[YOUR_USERNAME]/[YOUR_REPO_NAME].git

    cd [YOUR_REPO_NAME]


2. Grant execution permission:

     chmod +x *.sh


3. Run the master audit sequences:
   
    ./identity.sh
   
    ./inspector.sh
  
    ./auditor.sh
